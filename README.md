# 🌸 Iris Flower Prediction

> Machine Learning model to predict Iris flower species using Python and Scikit-learn

---

## 📋 Project Overview

| Item | Details |
|---|---|
| 🎯 Goal | Predict Iris flower species from measurements |
| 📊 Dataset | Iris Dataset (150 samples, 4 features) |
| 🤖 Algorithm | Random Forest, KNN, Logistic Regression |
| 🐍 Language | Python |
| 📦 Libraries | Pandas, Scikit-learn, Matplotlib, Seaborn |

---

## 🌺 About the Dataset

The Iris dataset contains measurements of 150 flowers from 3 species:
- 🌸 Iris Setosa
- 🌺 Iris Versicolor
- 🌼 Iris Virginica

### Features used for prediction:
| Feature | Description |
|---|---|
| Sepal Length | Length of the sepal (cm) |
| Sepal Width | Width of the sepal (cm) |
| Petal Length | Length of the petal (cm) |
| Petal Width | Width of the petal (cm) |

---

## 🔍 Project Steps

1. **Data Loading** — Load and explore the Iris dataset
2. **EDA** — Visualize distributions and correlations
3. **Preprocessing** — Encode labels, split train/test
4. **Model Training** — Train multiple ML models
5. **Evaluation** — Compare accuracy scores
6. **Prediction** — Predict species for new inputs

---

## 📈 Model Results

| Model | Accuracy |
|---|---|
| Random Forest | 97% |
| KNN | 96% |
| Logistic Regression | 95% |
